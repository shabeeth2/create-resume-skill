# 📄 create-resume

**An agent skill that builds ATS-friendly resumes in Markdown and exports them as DOCX (Word format) by default, with HTML + PDF available on request.**

## ✨ Features

- 💼 **Microsoft Word (DOCX) Output by Default** — Clean spacing, tabbed alignment, built with custom DOCX tables.
- 🎯 **ATS-optimised output** — Standard section names, clean hierarchy, no decorative elements.
- 📝 **Guided interview** — Agent collects your info one section at a time.
- 🌐 **Beautiful HTML & PDF** — Available as optional output flags with inline CSS and Iconify icons.
- 🔄 **Resumable** — Re-run anytime to update your existing resume.
- 🪶 **Self-contained** — No external services, no API keys, fully offline.
- 🖥️ **Cross-platform** — Windows, macOS, and Linux supported.

## 💬 Usage

Just ask your agent naturally. Any of these will trigger the skill:

```
"Create a resume for me"
"Help me write a CV"
"Format my work history into a resume"
"Update my existing resume"
"Export my resume to DOCX"
```

The agent will:
1. Check if a resume already exists and offer to update it
2. Collect your info one section at a time (name, experience, education, skills, etc.)
3. Generate `my-resume.md` in your project root
4. Convert it to `my-resume.docx` automatically (or `--html` / `--pdf` if explicitly requested)

---

## 📂 Output Files

| File | Description |
|------|-------------|
| `my-resume.md` | Editable Markdown source — the single source of truth |
| `my-resume.docx` | Microsoft Word Document (generated by default) |
| `my-resume.html` | Full HTML (generated on request via `--html` or `--pdf` flags) |
| `my-resume.pdf` | Headless PDF (generated on request via `--pdf` flag) |

All files are written to the **project root** (the directory where your agent is running).

## 📋 Requirements

| Requirement | Version | Notes |

| Python | 3.10+ | `python --version` to check |
| python-docx | 1.0+ | Auto-installed if missing |
| markdown-it-py | 3.0+ | Auto-installed if missing |
| Edge or Chrome | Any recent | For PDF export — optional |

Install Python dependencies manually if needed:

```bash
pip install markdown-it-py python-docx
```

## 🔧 Manual Conversion

You can run the converter directly without an agent:

```bash
# Convert to DOCX (Default)
python create-resume/scripts/convert.py my-resume.md

# Convert to DOCX + HTML
python create-resume/scripts/convert.py my-resume.md --html

# Convert to DOCX + HTML + PDF
python create-resume/scripts/convert.py my-resume.md --pdf

# Convert to HTML only (original behavior fallback)
python create-resume/scripts/convert.py my-resume.md --html-only

# Show version
python create-resume/scripts/convert.py --version
```

## 🛠️ Troubleshooting

### `python-docx` or `markdown-it-py` not found
```bash
pip install markdown-it-py python-docx
# or
pip3 install markdown-it-py python-docx
```

### PDF not generated
The PDF step requires Microsoft Edge (Windows) or Google Chrome (macOS/Linux) and must be invoked with `--pdf`.
Check the manual conversion troubleshooting steps in `README.md` if the auto-generator skips browser compilation.
